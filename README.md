## 📌 About The Project

This tool allows researchers, students, and biology enthusiasts to analyze DNA sequences instantly through an interactive web interface — no installation required.

Developed as part of a B.Tech Bioengineering portfolio project to demonstrate applied bioinformatics skills.

---

## ✨ Features

- 📊 *Nucleotide Composition Analysis* — counts of A, T, G, C bases
- 🧪 *GC & AT Content Calculation* — with interactive pie chart
- 🔬 *Molecular Biology Operations:*
  - Complement strand generation
  - Reverse complement
  - DNA → mRNA transcription
  - Protein translation (first ORF)
- 🧩 *Codon Frequency Analysis* — top 10 most frequent codons
- 🌡️ *Melting Temperature (Tm) Estimation* — Wallace Rule & Marmur-Chargaff Formula
- 📈 *Interactive Charts* — bar charts and pie charts
- 💡 *Example Sequences* — preloaded for quick testing
